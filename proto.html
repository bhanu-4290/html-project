EYE CONTROLLED WHEELCHAIR SYSTEM

The eye-controlled wheelchair system empowers tetraplegic individuals to maneuver independently. Using eye movements captured by a webcam, the system utilizes Gaze tracking with OpenCV for precise control. Pupil motion determines wheelchair direction, while blinking stops movement. Safety features include an ultrasonic sensor to detect obstacles and automatic halting, all managed by a cost-effective Raspberry Pi device.

SMART IRRIGATION SYSTEM USING IoT

The Smart Irrigation System using IoT is an innovative approach that leverages Internet of Things (IoT) technology to automate and optimize the irrigation process in agriculture. This system integrates various sensors like soil moisture sensors, humidity sensors, and temperature sensors to monitor the environmental conditions of the land. By connecting these sensors to a microcontroller like NodeMCU ESP8266, the system can collect real-time data and upload it to the cloud for analysis and monitoring.
